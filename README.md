🚀 **Meu Primeiro Projeto Spring Boot**
Este é um exercício que fiz na aula de Sistemas Distribuídos e Mobile deste semestre.
Na faculdade usamos Java, e usei o Spring Boot pra criar esse ecossistema de configurações
antes de criar minhas classes no Intellij.


🧠 **O que eu aprendi e apliquei:**
1. Arquitetura MVC (Model-View-Controller)
Implementei a lógica de navegação utilizando o padrão MVC. Onde o htmlController gerencia as rotas e decide qual "view" (HTML) entregar ao usuário.

2. Mapeamento de Rotas: Uso de @GetMapping para criar endpoints como /pagina

3. @Controller vs @RestController: Entendi na prática quando devolver dados brutos (JSON/String) e quando renderizar páginas completas.

4. Versionamento com Git: Fluxo completo de trabalho: init, add, commit e push.
   

🛠️ **Tecnologias Utilizadas**
Java 17+
Spring Boot 3.x
Maven (Gerenciador de dependências)
IntelliJ IDEA
Git & GitHub


**Como rodar o projeto localmente:**
1. Clone o repositório: git clone https://github.com/aliciaBalbino/primeiroSpringBoot.git
2. Abra no IntelliJ.
3. Execute a classe PrimeiroSpringBootApplication.
4. Acesse: http://localhost:8080/pagina3
